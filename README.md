# 안전자격증 단어카드 웹앱 (MVP)

순수 HTML/CSS/JavaScript로 구현한 **안전자격증 단어카드 공부용 웹앱**입니다.  
이미지 기반 단어 카드를 브라우저 LocalStorage에 저장해 두고, 카테고리별로 관리 및 학습할 수 있습니다.

## 폴더 구조

- `index.html` – 메인 HTML, 화면 구조 및 마크업
- `styles.css` – 모바일 우선 반응형 스타일, 카드/모달/오버레이 UI
- `app.js` – LocalStorage 연동, 카드 추가/삭제, 학습 모드 로직

## 실행 방법

1. 이 폴더 전체를 다운/복사합니다.
2. `index.html` 파일을 더블 클릭하거나 브라우저(Chrome 등)에서 열어 실행합니다.
3. 메인 화면에서 카테고리를 선택한 뒤, **관리** 버튼으로 카드를 추가/삭제하고, 카드 그리드에서 카드를 눌러 학습 모드를 사용합니다.

별도 빌드/번들 과정 없이 브라우저만 있으면 동작합니다.
